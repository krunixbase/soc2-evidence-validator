# soc2-evidence-validator

Modular validator for SOC 2 Type II evidence artifacts.
Includes control mapping, timestamp integrity, and audit-ready HTML reports.

## Deliverables
- Gap/evidence mapping (controls.json)
- CLI validator (validate.py)
- HTML report output (report.html)
